
# 🔒 Password Generator

A responsive, customizable **Password Generator** built using **React**, **Vite**, and **Tailwind CSS**.  
Quickly generate strong, random passwords with options for numbers, special characters, and adjustable password length.

---

## ✨ Features

- Fast & lightweight app built with **Vite**  
- Fully responsive, clean UI with **Tailwind CSS**  
- Adjustable password length via slider  
- Option to include **Numbers**  
- Option to include **Special Characters**  
- Real-time password generation  
- Dark theme design  

---

## 🚀 Tech Stack

- ⚛️ React  
- ⚡ Vite  
- 🎨 Tailwind CSS  
- 🛠️ JavaScript
